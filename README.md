# 球球对决

自动武器弹球模拟微信小程序第一版工程。

## 运行

1. 用微信开发者工具导入当前目录。
2. 将 `project.config.json` 里的 `appid` 改为你自己的小程序 AppID。
3. 开通微信云开发，并上传部署 `cloudfunctions/login`、`cloudfunctions/saveMatch`、`cloudfunctions/rooms`。
4. 在开发者工具中编译运行。

## 已实现

- 自由对战：未登录可选择模式、地图、2-4 个球，进入地图后点击放置小球，拖拽小球设置初速度方向和力度。
- 好友挑战：登录后创建房间、邀请好友，双方可进入地图布置自己的阵营，双方准备后进入同一配置对战。
- 历史战绩：登录后保存和读取战绩摘要。
- 战斗核心：固定 1/60s 步长、3 种地图、4 种模式、24 个配置化球体、确定性 seed、解卡碰撞、活跃轨道偏置、结算摘要。
- 表现层：Canvas 几何绘制 24 个球的主体、图标、武器、投射物、陷阱、轨迹，并用程序合成占位音效触发 SFX 事件。
- 专业美术层：`miniprogram/assets/balls/` 内置 24 张 256x256 透明 PNG 球体 sprite，战斗和图鉴优先使用 PNG，缺图时才回退到 Canvas 几何绘制。
- 当前小球碰撞半径已整体翻倍，视觉直径与碰撞体同步放大。
